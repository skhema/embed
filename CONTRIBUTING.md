# Contributing

This repository (`skhema/embed`) is an **automatically generated, read-only
mirror** of `@skhema/embed`, which is developed in Skhema's private monorepo.

- **Pull requests are disabled.** Any change pushed here is overwritten on the
  next release, so we cannot accept PRs against the mirror.
- **The source of truth is private.** Bug reports, feature requests, and
  questions are welcome at [support@skhema.com](mailto:support@skhema.com).
- **Security reports:** see [SECURITY.md](./SECURITY.md).

Thank you for your interest — routing feedback through the address above lets us
fold it into the upstream package.
